What do we do?

Every developer has a few certain tasks that they repeat for every project or even feature that they work on. Rachis merges those common tasks, code, and 3rd party APIs into a single, easy to use, serivce. Whether you need to spin up a database, a social login, need to know the weather for a user's location, need to create a playlist, or countless other common features, we've got you covered.

How it works

Configure your endpoint on rachis

Our library of endpoints and integrations is continously growing. Configure and save your unique API endpoint.

Use any language to make HTTP requests to your endpoint

Our API is language agnostic. We have no SDK's, we operate exclueisvley through HTTP requests. If your app can talk to the internet, you can talk to the Rachis API. 

var request = require('request');

request('http://api.rachis.io/?point=590fe912060721e206f3d74d', function(error, response, body) {
    console.log(body);
});
